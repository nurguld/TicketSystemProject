## 📌 Overview
A web-based ticket and admin management system built with **ASP.NET MVC** and **PostgreSQL**.  
It provides secure CRUD operations, role-based authorization, and a user-friendly interface.

## 🚀 Features
- Role-based authorization (Admin / User)
- Secure Create, Read, Update, Delete operations
- PostgreSQL integration with EF Core
- Responsive UI with Razor & Tag Helpers
